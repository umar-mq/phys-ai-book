Capstone Project Guide: The Physical AI Agent

Welcome to the Finaleâ€‹

Congratulations on reaching the end of the Physical AI course. This Capstone Project is not just a test; it is a portfolio piece. It demonstrates that you can integrate:

  1. Mechanical Simulation (Isaac Sim)
  2. Robot Middleware (ROS 2)
  3. Generative AI (LLMs & VLMs)

The Missionâ€‹

"Design and deploy an autonomous agent capable of Semantic Exploration and Manipulation."

In simple terms: The robot must explore a room, find objects requested by a human, and interact with them.

Hardware Tracksâ€‹

You can choose one of two tracks depending on your budget:

Track A: The Simulation Specialist (Pure Software)

  • Requirement: A powerful PC (RTX GPU).
  • Task: Create a highly complex environment in Isaac Sim (e.g., a hospital room). The robot (NurseBot) must find medicine and bring it to the patient bed.
  • Bonus: Add "dynamic actors" (people walking) that the robot must avoid.

Track B: The Hardware Hacker (Sim-to-Real)

  • Requirement: Jetson Orin Nano + Robot Arm/Mobile Base.
  • Task: A simpler environment (Tabletop). The robot must sort objects (Red blocks to Red bin, Blue blocks to Blue bin) based on voice commands.
  • Focus: Dealing with real-world noise and lighting.

Submission Requirementsâ€‹

  1. GitHub Repository:
    • src/: Your ROS 2 packages.
    • launch/: One main launch file capstone.launch.py.
    • docs/: Architecture diagram (Mermaid/Draw.io).
  2. Video Demo (3 Minutes):
    • Show the Voice Command.
    • Show the "Thinking Process" (Terminal output of LLM).
    • Show the Execution.
    • Show one "Failure & Recovery" case (e.g., move the object while the robot is reaching).

Grading Rubricâ€‹

ComponentPointsDescription
Architecture30Clean ROS 2 graph, Proper usage of Actions/Services, TF Tree correctness.
AI Integration30Successful use of LLM for planning and VLM/YOLO for vision.

Robustness20Does it crash if I turn off the lights? Does it recover from empty grasps?
